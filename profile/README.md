이곳은 코드크래프트의 땀과 노력이 깃든 신성한 코드 저장소입니다.

신성한 곳을 방문할때의 규칙은 아래와같습니다. 

1. 목욕재개후 마우스와 키보드를 항균티슈를 깨끗이 닦는다.

2. 미지근한 중성세제를 이용해 모니터를 닦는다.

3. 나쁜생각과 세상의 모든 근심걱정을 떨친후 조심스러운 마음으로 접근한다.

4. 소원을 빈뒤 코드크래프트의 계좌에 소원의 크기만큼 현금을 입금한다.
